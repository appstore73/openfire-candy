Candy Plugin Readme

Prerequesites

Installation

  1. Install the clientcontrol plugin.
  2. Stop Openfire.
  3. Copy the candy.war file to the OPENFIRE_HOME/plugins directory.
  4. Restart Openfire.
  5. From a browser, go to http://your_openfire-server:7070/candy
  6. If this page does not appear, please check you log files and post any errors on www.igniterealtime.org

Configuration

Normal

  • Follow steps from https://github.com/candy-chat/candy/wiki/Installing-a-Jabber-server

Advanced
